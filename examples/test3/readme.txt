problem.c: the problem definition source code.
	   You will modify this code to set up model scenario, add additional physics, and control the output style. Compile the rest of the rebound code with this one to obtain the 'rebound' executable to perform your simulation(s).

trajectory.m: post-processing Matlab script to analyze and visualize the data.

r2e.m: Matlab script to convert the orbital position and velocity to the six orbital elements.

ReadParticle.m: Matlab script to read the binary particle data.
